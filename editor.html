📖 TBM 매뉴얼 편집기

매뉴얼 화면을 그대로 보면서 글자를 클릭해 바로 고치는 도구입니다. 코딩 지식이 필요 없습니다.

  1. 매뉴얼 열기를 눌러 index.html을 선택합니다. (이 파일과 같은 폴더에 있습니다)
  2. 고치고 싶은 문장을 클릭하고 그냥 타이핑합니다. 실수하면 Ctrl+Z.
  3. 같은 단어를 문서 전체에서 바꾸려면 용어 일괄 치환을 사용합니다.
  4. 저장(Ctrl+S)을 누르면 index.html에 바로 반영됩니다.
💾 Chrome·Edge에서는 파일에 바로 저장됩니다. 처음 저장할 때 원본 백업본이 자동으로 다운로드됩니다.
다른 브라우저에서는 수정본이 다운로드되며, 받은 파일로 기존 index.html을 교체하면 됩니다.

⚙ PDF 페이지 전환 설정

체크한 항목은 PDF 저장/인쇄 시 새 페이지에서 시작합니다. 각 장(0~5, 부록)은 항상 새 페이지에서 시작하며, 이 설정은 화면 표시에는 영향이 없습니다. 변경 후 저장(Ctrl+S)해야 파일에 반영됩니다.

체크 변경은 즉시 문서에 표시되며, 저장 시 파일에 기록됩니다.
