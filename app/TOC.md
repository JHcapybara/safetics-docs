# TBM 모바일 앱 사용 설명서 — 목차 초안

> 대상: 비개발자 (영업, 마케팅, 운영, 총무, 관리직 등) + 현장 근로자
> 원칙: 메뉴 순서가 아닌 **업무 시나리오 순서**. 각 항목은 "목적 1줄 → 스크린샷 → 번호 단계 → 주의/팁" 패턴, 1작업 1페이지.
> 용어: 앱 UI 문구와 100% 동일하게 사용 (`lang/ko/common.json` 기준). 혼용 용어는 첫 등장 시 병기
> (예: "건의사항(민원)", "인증번호 로그인(외부 근로자)", "위치 인증(Geo-Fence)").

---

## 0. 들어가기 전에 — 핵심 개념 5분 정리

| 개념 | 한 줄 설명 |
|---|---|
| TBM (Tool Box Meeting) | 작업 전 안전회의. 관리자가 웹에서 발송하면 근로자가 **앱으로 참여** |
| 웹 ↔ 앱 역할 구분 | **발송·관리는 관리자 웹**, **참여·신고는 이 앱** (0장에서 표로 명확히 구분) |
| 체크리스트 | TBM 1건에 담긴 확인 항목 묶음: 주의사항 / 개인안전보호구(PPE) / 오늘의 특별 공지 |
| 확인하였음 / 이슈 있음 | 항목별 응답 2종. "이슈 있음"은 관리자에게 TBM 이슈로 자동 접수 |
| 서명 방식 3종 | 자필 서명 / 위치 인증(Geo-Fence) / 확인만(서명 없음) — 관리자가 발송 시 결정 |
| 정규직 / 외부 근로자 | 정규직 = 아이디(이메일) 로그인. 외부 근로자 = 공장 게시 인증번호 + 이름 + 휴대폰 번호 |
| 파견 | 다른 조직 현장의 TBM 대상으로 초대받는 것. 앱에서 수락/거절 |
| 건의사항 | 위험원 보고·작업 중지·SOS·건의·TBM 이슈가 모두 모이는 곳. 조치 이력을 답글로 소통 |
| TBM 상태 3종 | 예정 / 완료 / 미완료 |
| 조치 상태 3종 | 미조치 → 조치중 → 조치 완료 |

## 1. 시작하기

### 1.1 설치와 준비물
- 앱 설치(스토어), 정규직: 계정(SafetyDesigner와 동일), 외부 근로자: 현장 게시 인증번호
- 권한 안내: 위치(위치 인증·날씨), 카메라·사진(신고 첨부), 알림(Push)

### 1.2 로그인 — 정규직 (아이디 로그인)
- 아이디(이메일)/비밀번호, "로그인 상태 유지"(자동 로그인), 언어 전환(KO/EN)
- 오류: "아이디 또는 비밀번호가 올바르지 않습니다." 등
- 화면: `app/index.tsx` (탭 "아이디 로그인")

### 1.3 로그인 — 외부 근로자 (인증번호 로그인)
- 인증번호 + 휴대폰 번호 + 이름 → 본인 확인(생체인증 필수 등록 또는 PIN 4자리)
- PIN 잊음 → "관리자에게 인증 정보 초기화 요청"
- 화면: `app/index.tsx` (탭 "인증번호 로그인"), PinDialog

### 1.4 생체인증·자동 로그인
- 정규직: 로그인 후 생체인증 등록 제안(선택) → 다음부터 "생체인증 로그인" 버튼
- 외부 근로자: 매 로그인 본인 확인 필수, 자동 로그인 없음
- 화면: AuthContext, biometricService

### 1.5 화면 구성 익히기 (홈 · 메뉴)
- 홈: 현장 날씨 / 오늘의 TBM / 긴급 카드 3종(위험원 보고·SOS·작업 중지) / 건의사항 요약
- 오른쪽 메뉴(드로어): 홈 / Tool Box Meeting / 건의사항 / 설정 / 언어 / 로그아웃
- 화면: `app/(main)/index.tsx`, MainDrawer

### 1.6 현장 선택 (여러 현장 소속일 때)
- 홈 상단 현장 이름 ▼ 탭 → "현장 선택" 바텀시트 → 파견 현장은 "파견" 배지
- 선택 시 TBM·건의사항이 해당 현장 기준으로 필터
- 화면: SiteContext, 홈 헤더 / 설정 화면에서도 변경 가능

## 2. 주요 기능 — 매일 하는 일 (TBM 참여) ★

### 2.1 오늘의 TBM 확인하고 시작하기
- 홈 "오늘의 TBM" 카드 또는 Tool Box Meeting 탭 → TODAY 카드 → "TBM 시작하기"
- 시작 전이면 "{시각}부터 참여할 수 있습니다"
- 세션 요약 화면: 작업 책임자·현장·참석 인원·체크리스트 구성(건수) → "시작하기"
- 화면: `app/(main)/tbm.tsx`, `app/tbm/session/[id].tsx`

### 2.2 체크리스트 응답하기 (확인하였음 / 이슈 있음)
- 항목 1장씩: 위험인자/유해요인 태그 + 설문 내용 확인 → "확인하였음" 또는 "이슈 있음"
- "이슈 있음" → 관리자에게 TBM 이슈 자동 접수
- 이전/다음 페이지 이동, 응답해야 다음 진행
- 화면: `app/tbm/checklist/[id].tsx`

### 2.3 참여 완료 — 자필 서명
- 요약(전체/확인/이슈) 확인 → "서명하기" 영역에 손가락 서명 → "다시 서명" 가능 → "TBM 완료하기"
- 이슈가 있으면 버튼이 "이슈 보고 및 서명 제출"로 변경
- 화면: SignPage (checklist 내)

### 2.4 참여 완료 — 위치 인증(Geo-Fence)
- 자동으로 "위치 확인 중…" → 성공 "위치 인증 완료" → "TBM 완료하기"
- 실패: "현장 반경 밖에 있습니다. (Nm / 허용 Nm)" → "다시 시도"
- 위치 권한/GPS 오류 대처
- 화면: GeofenceVerifyPage (checklist 내)

### 2.5 지난 TBM 기록 확인
- Tool Box Meeting 탭 → 이전 기록 → 필터(전체/완료/미완료/예정)
- 기록 상세: 항목별 "확인 완료/이슈 있음", 특별 공지, 참석 확인 서명(또는 "위치 인증으로 참여 완료")
- 화면: `app/tbm/[id].tsx`

## 3. 부가 기능 — 가끔 하는 일

### 3.1 건의사항 작성하기
- 건의사항 탭 → ✏ 버튼 → "새 건의": 제목(필수)·내용·사진 첨부(최대 3장) → "등록"
- 화면: `app/(main)/suggestions/new.tsx`

### 3.2 건의사항 조회·답글 달기
- 목록: 검색·필터 칩, 상태(미조치/조치중/조치 완료)
- 상세: 조치 이력(채팅형), 답글 입력·수정·삭제 (조치 완료 건은 답글 마감)
- 화면: `app/(main)/suggestions/index.tsx`, `[id].tsx`

### 3.3 파견 초대 수락/거절
- 로그인 직후 "파견 초대" 모달 또는 설정 화면 "파견 초대 (N)" 섹션
- 카드: 초대 현장·기간(즉시~무기한)·메모 → "수락" / "거절"
- 수락 시 현장 선택 목록에 파견 현장 추가
- 화면: DispatchContext, 홈 모달, profile

### 3.4 알림 설정
- 설정: "App Push 알림 받기" / "TBM 등록 알림 받기" 토글
- 화면: `app/(main)/profile.tsx`

### 3.5 언어 변경 (한국어/English)
- 메뉴(드로어) > 언어 → 한국어(기본) / English. 로그인 화면에서도 KO/EN 전환
- 화면: MainDrawer, LangContext

### 3.6 로그아웃·계정 삭제
- 메뉴 > 로그아웃 (확인 모달)
- 설정 > 계정 삭제 — 복구 불가 경고
- 화면: MainDrawer, profile

## 4. 문제 생겼을 때 — 긴급 신고와 문제 해결 ★

### 4.1 위험원 보고
- 홈 긴급 카드 "위험원 보고" → 제목·내용·사진 → "등록" → 건의사항에서 조치 추적
- 화면: `suggestions/new?from=hazard`

### 4.2 긴급 SOS
- 홈 "SOS" 카드 → 확인 대화상자("예, 전송합니다") → 사이렌+진동 → "SOS 종료" 시 관리자에게 전송
- 전송 메시지: "[긴급 SOS] 현장에서 긴급 구조 요청이 발생했습니다..."
- 화면: `app/emergency/sos.tsx`

### 4.3 작업 중지 요청
- 홈 "작업 중지 요청" 카드 → 사유 선택(24종 + 직접 입력) → 상세 내용·사진(선택) → "작업 중지 요청 전송"
- 화면: `app/emergency/stop.tsx`

### 4.4 로그인이 안 될 때
- 오류 메시지별 원인·해결 표 (계약 만료, 조직 없음, 인증번호 오류, 근로자 미등록 등)

### 4.5 위치 인증(Geo-Fence)이 안 될 때
- 위치 권한 / GPS / 현장 반경 밖 — 각 메시지별 대처
